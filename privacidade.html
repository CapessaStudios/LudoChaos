Resumo direto

O LudoChaos não pede contas, não pede email, não usa anúncios e não vende dados a ninguém. Esta é a explicação completa de como o jogo trata as tuas informações.

Sem contas Sem anúncios Sem rastreamento Sem venda de dados Guarda progresso no telemóvel Multijogador opcional

1. Dados guardados só no teu dispositivo

Para jogares offline (contra bots ou pass-and-play), o LudoChaos guarda tudo localmente no teu telemóvel através do armazenamento da aplicação. Isto inclui:

  • Progresso do jogo em curso (posições das peças, cartas na mão)
  • Estatísticas, conquistas e sequência de vitórias
  • Chaos Coins e cosméticos comprados na loja (peças, tabuleiros, dados, efeitos)
  • Preferências (tema claro/escuro, volume, animações de fundo)

Estes dados nunca saem do teu telemóvel nesta forma de jogo. Não temos acesso a eles, não os copiamos para servidores. Se desinstalares a aplicação ou limpares os dados, desaparecem completamente.

2. Dados usados no multijogador online

Quando escolhes jogar online (criar ou entrar numa sala), o LudoChaos utiliza o Firebase Realtime Database (Google) para sincronizar a partida entre jogadores.

Para isso, alguns dados temporários podem ser partilhados:

  • Nome/alcunha escolhido antes de entrar na sala.
  • Avatar ou emoji selecionado pelo jogador.
  • Código da sala e estado da partida (dados lançados, posições das peças e cartas utilizadas).

Estes dados existem apenas enquanto a sala está ativa. Quando a partida termina ou todos saem, a sala é removida. Não pedimos email, número de telefone, localização, câmara ou contactos.

3. Notificações

Se autorizares, o LudoChaos pode enviar notificações locais no teu telemóvel relacionadas ao funcionamento do jogo.

Podes desativar esta opção a qualquer momento nas definições do teu dispositivo. As notificações nunca são utilizadas para publicidade.

4. O que NÃO fazemos

  • Não mostramos anúncios de terceiros.
  • Não usamos ferramentas de análise ou rastreamento (Google Analytics, Meta, etc.).
  • Não vendemos, alugamos ou partilhamos dados para fins comerciais.
  • Não pedimos acesso à câmara, microfone, contactos ou localização.

5. Serviços de terceiros

O único serviço externo utilizado é o Firebase (Google) , exclusivamente para permitir o funcionamento do multijogador online e sincronizar as partidas entre dispositivos.

A Google pode processar alguns dados técnicos conforme a sua própria política de privacidade, como fornecedora da infraestrutura. A Capessa Studios não utiliza esses dados para publicidade ou venda.

6. Crianças

O Ludo Chaos é um jogo de tabuleiro casual adequado para diferentes idades. Não recolhemos informações pessoais identificáveis de crianças ou outros utilizadores.

Não existe criação obrigatória de conta, recolha de dados sensíveis ou sistemas de interação que exijam informações pessoais.

7. Segurança das informações

A Capessa Studios procura utilizar boas práticas para proteger a experiência dos jogadores.

  • Dados offline permanecem armazenados no próprio dispositivo.
  • Dados temporários do modo online são utilizados apenas para sincronizar partidas.
  • Não recolhemos palavras-passe, documentos ou informações privadas.

8. Eliminação de dados

Como grande parte dos dados do jogo é armazenada localmente no dispositivo, o jogador pode remover essas informações a qualquer momento.

  • Apagar os dados da aplicação remove progresso, configurações e compras locais.
  • Sair de uma sala online remove a participação daquela partida.
  • Desinstalar o aplicativo remove os dados guardados no dispositivo.

9. Alterações nesta política

A Capessa Studios pode atualizar esta Política de Privacidade para refletir melhorias no jogo, novos recursos ou alterações nos serviços utilizados.

Quando houver alterações importantes, a nova versão será publicada nesta página com uma nova data de atualização.

10. Serviços externos utilizados

Para fornecer algumas funcionalidades, como o modo multiplayer online, o Ludo Chaos pode utilizar serviços externos de infraestrutura.

  • Firebase (Google) — utilizado para sincronização das partidas online.
  • Serviços de armazenamento temporário necessários para funcionamento online.

Esses serviços possuem as suas próprias políticas de privacidade. A Capessa Studios não vende nem utiliza esses dados para publicidade.

11. Responsabilidade do utilizador

Ao utilizar o Ludo Chaos, o jogador concorda em utilizar o aplicativo de forma adequada e respeitar as funcionalidades disponibilizadas.

O jogo foi desenvolvido para oferecer entretenimento, competição saudável e uma experiência divertida entre jogadores.
